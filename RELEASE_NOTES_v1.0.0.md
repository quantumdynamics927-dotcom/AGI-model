```bash

cd TMT-OS
cp .env.example .env
# Edit .env with your IBM Quantum token
docker-compose up -d

# Access services:
# API: http://localhost:8000
# Dashboard: http://localhost:8501
# Grafana: http://localhost:3000
```

### Local Development
```bash
git clone https://github.com/quantumdynamics927-dotcom/TMT-OS.git

source tmt-env/bin/activate  # Windows: tmt-env\Scripts\activate
```
